  • Simple Brew Tool

    There are a lot of log programs and recipie formulation programs that exist. Most contain features that are beyond what I need as a weekend homebrewer or lack features that I'm looking for. Simple Brew Tool is designed to address those issues in a flexible way. I'm writing the whole system in ClojureScript using the Om framework.
